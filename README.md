# CmsBundle

CMS administrable AkyosCMS : pages, menus, utilisateurs, SEO, RGPD, champs personnalisés, options et gestion des accès admin. S’appuie sur CoreBundle et BuilderBundle.

## Compatibilité

- PHP >= 8.4
- Symfony ^8.0

## Dépendances Akyos

- `akyos/core-bundle` ^8.0
- `akyos/builder-bundle` ^8.0

## Installation

```bash
composer require akyos/cms-bundle:^8.0
```
